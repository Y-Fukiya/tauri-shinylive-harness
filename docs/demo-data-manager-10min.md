# 10-Minute Demo: Data Manager

## Audience

Data manager, clinical programmer, QA reviewer.

## Message

The harness can demonstrate data consistency checks and query-candidate style
evidence without importing production clinical data.

## Flow

1. Open `reports/clinical-data-pack-validation.json`.
2. Show required domain files and required columns.
3. Show subject ID reference checks.
4. Show visit reference checks.
5. Show controlled terminology checks.
6. Show AE start/end day checks.
7. Show medication indication alignment checks.
8. Show issue summaries by severity, subject, domain, and rule.
9. Open `docs/generated/clinical-data-dictionary.md`.
10. Open `reports/evidence-index.html`.

## Talk Track

This is a synthetic data quality walkthrough. It can help teams decide which
checks and summaries are useful before they design a production workflow.
