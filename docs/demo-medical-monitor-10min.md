# 10-Minute Demo: Medical Monitor

## Audience

Medical monitor, clinician, clinical scientist.

## Message

Offline synthetic subject profiles can be used to evaluate review flow, visual
context, and report output without exposing patient data.

## Flow

1. Open the portal and select Subject Profile Reference App.
2. Select `SUBJ-001`.
3. Review demographics and study status.
4. Review the visit timeline.
5. Review ALT/AST lab trends.
6. Compare AE intervals with exposure intervals.
7. Open the Subject Snapshot report.
8. Point to app version, generated timestamp, data pack ID, and data pack hash.
9. Point to the clinical-use limitation.
10. Close by opening `reports/evidence-index.html`.

## Talk Track

This is a sandbox for review workflow evaluation. It is useful for discussing
whether the subject-level review design is clear, not for making clinical
decisions.
