# Subject Profile Scenario

## Intended Audience

- Medical monitor
- Safety reviewer
- Clinical scientist
- Data manager

## What This Scenario Demonstrates

- Subject-level timeline review
- Lab trend review
- AE context review
- Exposure and AE interval comparison
- Data pack hash traceability
- HTML and PDF report export

## What This Scenario Does Not Demonstrate

- Real patient review
- Clinical decision support
- Full protocol-specific medical monitoring
- Submission-ready SDTM or ADaM
- Part 11 electronic records or signatures
