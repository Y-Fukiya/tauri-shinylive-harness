# Medical Monitor Quickstart

## Goal

Evaluate whether a synthetic subject profile app supports a useful medical
review walkthrough without using real patient data.

## Try This

1. Open the diagnostics portal.
2. Launch the Subject Profile Reference App.
3. Select `SUBJ-001`.
4. Review the visit timeline.
5. Review lab trends, especially ALT/AST.
6. Compare AE timing with exposure timing.
7. Open the Subject Snapshot report.
8. Confirm the data pack hash is visible.

## Evaluate

- Can you understand the subject story quickly?
- Are lab abnormalities and AEs easy to compare?
- Is exposure timing visible enough for review discussion?
- Does the report contain enough context for a walkthrough?
- Are the synthetic-data and non-clinical-use boundaries clear?
